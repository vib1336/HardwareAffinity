# HardwareAffinity
Online electronics store

## Build with:

* ASP.NET Core 3.1
* EF Core 3.1
* SQL Server 2017 Express
* Vanilla JavaScript/jQuery
* Ajax
* Cloudinary
* SendGrid
* Font Awesome
* Moment.js
